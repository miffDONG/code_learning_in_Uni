인적사항
'나' 소개 항목 리스트:


  • 이름 : 김현동

  • 나이 : 23세

  • 학교 : 한국외국어대학교

  • 학과 : 컴퓨터공학부

  • 이력 : UMC server.Spring.github

    연합동아리 UMC 스프링 백엔드 부분에서 활동했습니다.
    협업의 경험도, 학구열도 부족하다고 생각했기에 다른 사람과의 접촉이 필요했습니다.
    다른 사람은 어떻게 공부하는지, 나와의 격차는 얼만큼인지 확인할 수 있었고 프로젝트 협업을 진행하면서
    내가 얼마나 부족하고 어떤걸 학습해 나가야할지 알 수 있었습니다.


  • 개인공부 : github 기록

  • 취미 : 그림

    • 마음이 복잡할때 집중할 수 있는 것을 찾고는 하는데, 그중 하나가 그림입니다. 종이와 연필만 있으면 되기에 어디에서도 할 수 있고
    그림 그릴때만큼은 다른곳에 신경을 안 쓸 수 있어서 혼란스러울때 주로 그립니다.



  • 특기 : SUP(Standup paddleboarding)

    • 파도 없이 잔잔한 한강 가운데에서 여유를 즐길 수 있어서 시작했습니다. 평상시에는 주변, 하늘을 잘 보지 않는데
    이곳에서는 아름다운 풍경이 펼쳐져 있어 감탄을 하며 SUP의 매력에 빠졌습니다.
    그러다가 대회가 개최된다는 소식을 들었고 바다에서도 즐길 수 있는 스포츠이기에 열심히 타기 시작했습니다.
    최근 9.24일 한국에서 처음 개최된 국제대회 SUP 일반부 1KM대회에서 참여하고 왔습니다.
    좋은 성과를 얻지는 못했지만 세계에 얼마나 많은 멋진사람들이 있는지,
    내가 이 SUP를 얼마나 좋아하는지 알게된 경험이였습니다. 다음 대회개최 전까지 열심히 탈 예정입니다.
